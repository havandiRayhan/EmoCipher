🔐 EmoCipher: Emoji Cryptography for Kids

EmoCipher adalah sebuah aplikasi mobile edukatif yang dirancang khusus untuk mengajarkan konsep dasar kriptografi kepada anak-anak dengan cara yang menyenangkan dan interaktif. Aplikasi ini mengubah teks biasa menjadi deretan emoji (enkripsi) dan sebaliknya (dekripsi).

Proyek ini dibangun menggunakan Python dan Kivy, dengan fokus utama pada perancangan User Interface (UI) dan User Experience (UX) yang ramah anak (kid-friendly).

✨ Fitur Utama

Emoji Encryption: Mengonversi pesan teks menjadi kombinasi emoji yang unik.

Emoji Decryption: Mengembalikan deretan emoji rahasia menjadi teks yang dapat dibaca.

Kid-Friendly UI/UX: Antarmuka yang cerah, tombol berukuran besar, dan navigasi yang sangat intuitif agar mudah dioperasikan oleh anak-anak.

Android Ready: Proyek ini dapat di-compile menjadi format APK untuk dijalankan di perangkat Android.

🛠️ Teknologi yang Digunakan

Bahasa Pemrograman: Python 3.x

Framework GUI: Kivy

Packaging: Buildozer (untuk kompilasi APK)

🚀 Cara Menjalankan Aplikasi Secara Lokal

Jika Anda ingin menjalankan aplikasi ini di komputer Anda:

Pastikan Anda telah menginstal Python.

Clone repositori ini:

git clone [https://github.com/havandiRayhan/EmoCipher.git](https://github.com/havandiRayhan/EmoCipher.git)


Masuk ke direktori proyek:

cd EmoCipher


Instal library yang dibutuhkan:

pip install -r requirements.txt


Jalankan aplikasi:

python main.py


🧠 Konsep Edukasi

Aplikasi ini memperkenalkan konsep dasar penyandian data (enkripsi substitusi sederhana) di mana setiap karakter diwakili oleh simbol (emoji) tertentu. Tujuannya adalah untuk memancing ketertarikan anak-anak terhadap keamanan digital dan computational thinking sejak dini.
